Afinamento

Reduz estruturas até seus esqueletos centrais, ideal para medição fina.

Dilatação

Expande objetos em imagens binárias, útil para conexões quebradas.

Erosão

Remove pixels das bordas dos objetos, útil para eliminar ruídos.

Laplaciano

Destaca bordas usando segunda derivada, útil em imagens com contraste suave.

Limiar

Converte tons de cinza em preto e branco com base em um intervalo definido.

Média

Aplica desfoque suave para reduzir ruídos, mantendo detalhes principais.

Sobel X

Detecta bordas verticais, ideal para identificar linhas horizontais.

Sobel Y

Detecta bordas horizontais, ideal para destacar estruturas verticais.

Sobel XY

Combina Sobel X e Y para detecção completa de bordas em qualquer direção.

Mensuração

Mede áreas e diâmetros de estruturas anatômicas automaticamente.

Gráfico

Visualize os dados coletados com gráficos interativos e comparativos.

Ajustes Finos

Refina os parâmetros de análise e segmentação para resultados mais precisos.
